# 完成頁促銷圖片

擴充完成頁的推薦區塊會用到這裡的圖片。檔名要跟 `_build.py` 的 `promo_json()` 對應：

| 檔名 | 內容 | 對應項目 |
|---|---|---|
| `stay-shermuh.jpg` | 阿里山神木賓館 | 住宿 1 |
| `stay-hofong.jpg`  | 禾楓別墅 | 住宿 2 |
| `tour-sunrise.jpg` | 阿里山日出／神木行程 | 行程 |

規格：
- **單張 < 400KB**（超過擴充會略過該圖，只顯示文字，不會壞）。
- 建議寬 ≥ 600px、橫幅比例（約 16:9 或 3:2）。
- 用你有權使用的圖（自己拍的、或聯盟後台提供的官方素材）。

要改促銷「文字／連結」→ 改 `_build.py` 的 `promo_json()` 再跑 `python _gen.py`。
要改「圖片」→ 直接換這裡的檔案（檔名不變就不用重跑產生器）。
